# ReserveForClass 

1)**Contexto do Projeto**:
    O projeto **ReserveForClas** teve como objetivo desenvolver uma plataforma prática e organizada para permitir que professores e coordenadores realizem a reserva de Chromebooks em Escolas para uso em sala de aula e Bibliotecas. A proposta busca otimizar a disponibilidade dos dispositivos, evitar conflitos de horários e melhorar o planejamento das atividades escolares que dependem de tecnologia.

2) **Passo a Passo para Instalação**:
    Fazer a instalação do Visual Studio Code (VS code), dentro Terminal do VS code, fazer o git clone do Repositório https://github.com/Ferreiraxzk/ReserveForClass.git dentro do terminal (BASH) do VS code, Após isso, entre dentro da pasta do Repositório e logo em seguida instale as bibliotecas contidas dentro do requirements.txt com o comando pip install -r requirements.txt no terminal, criar uma venv (Ambiente Virtual) para fazer a aplicação iniciar sem erros, a diante executar o comando python app.py no terminal e seguir o link em que o terminal irá apresentar.
